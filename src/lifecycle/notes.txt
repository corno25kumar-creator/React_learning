## 1Ô∏è‚É£ Mounting ‚Äì When component is created

This is when the component appears on the screen for the first time.

### Important methods (Class Component)

Used to:

* Set initial state
* Bind methods


üü¢ Use for: initial setup

# Lifecycle in Functional Components (Hooks)

Now React mostly uses **functional components with Hooks**.

## useEffect = Lifecycle in Hooks

### On Mount

```js
useEffect(() => {
  fetchData();
}, []);
```

### On Update

```js
useEffect(() => {
  console.log("count changed");
}, [count]);
```

### On Unmount (Cleanup)

```js
useEffect(() => {
  return () => {
    console.log("component removed");
  };
}, []);
```

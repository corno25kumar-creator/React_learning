### âœ… Definition 

A **Custom Hook** in React is a JavaScript function whose name starts with `use` and which allows you to **reuse stateful logic** across multiple components using built-in React Hooks.

# ðŸ”¸ When Should You Create a Custom Hook?

Create a custom hook when:

* The same logic is used in **multiple components**
* The logic is related to **state or side-effects**
* You want to keep components clean and focused on UI

### Example use cases:

* Fetching data from APIs
* Form validation
* Debouncing inputs
* Authentication logic
* Window resize detection
* Theme management
* Online/offline detection

# ðŸ”· Real-World Custom Hook Example: useFetch

### ðŸŽ¯ Problem:

Many components fetch data using Axios/fetch.

### âŒ Without Custom Hook

```jsx
useEffect(() => {
  fetch(url).then(...)
}, []);
```

Repeats everywhere.

---

---------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------


### âœ… With Custom Hook

```js
import { useEffect, useState } from "react";

function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetch(url)
      .then(res => res.json())
      .then(data => setData(data))
      .catch(err => setError(err))
      .finally(() => setLoading(false));
  }, [url]);

  return { data, loading, error };
}
```

### Use in component

```jsx
const { data, loading, error } = useFetch("https://api.com/users");
```

# ðŸ”· Custom Hook for Window Resize

```js
function useWindowSize() {
  const [size, setSize] = useState(window.innerWidth);

  useEffect(() => {
    const handler = () => setSize(window.innerWidth);
    window.addEventListener("resize", handler);
    return () => window.removeEventListener("resize", handler);
  }, []);

  return size;
}
```
